mediaTypes

mediaTypes: string[] = ['application/jats+xml']

Overrides Codec.mediaTypes

  • Defined in src/codecs/jats/index.ts:61

Media types that this codec will match.

There is currently no registered media type for JATS.

This custom type uses the convention of using application/ and the +xml suffix to differentiate this format from other XML-based formats.

    Sniff content to see if it is JATS XML.

    For speed, rather than parsing the entire content as XML, this function just uses a regex to checks if the content contains a JATS DOCTYPE declaration. Obviously, it's possible that a file in some other format (e.g. Markdown) could contain this string.
