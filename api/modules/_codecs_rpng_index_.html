Styling to be applied to a node's HTML fragment when generating the rPNG.

This is decoupled from any themes because rPNGs are usually used in a context where themes do not apply e.g. in a docx file.

Currently, rather than try to do some fancy CSS module loading, we just copy and paste this file in here.
